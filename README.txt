== Description ==

Restrict the content of all WordPress, just select the roles that can access that post, page, custom post type, product ... etc.

A very lightweight plugin to restrict posts even WooCommerce products.

Not much more to say, leave your review :)

== Installation ==

1. Upload the plugin files to the /wp-content/plugins/plugin-name directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the ‘Plugins’ screen in WordPress
3. Enjoy!
 
== Frequently Asked Questions ==

= How configure =

On each post, page or custom post type an option is available to select the roles that will have access to that post, page, or custom content type.

= Work with WooCommerce? =

YES!
